Multihead Approximators
=======================

This section documents the approximators designed for multihead attention modules within the `multihead` subfolder. These approximators focus on approximating the multihead attention mechanism, offering both fixed and trainable approximations.

Customizable MultiHead Approximator
-----------------------------------

.. automodule:: src.hela.approximation.approximators.multihead.customizable_multihead
   :members:
   :undoc-members:
   :show-inheritance:
   :special-members: __init__

The ``CustomizableMultiHeadApproximator`` handles the approximation of the multihead attention module. It allows for customization of the attention mechanism and provides both trainable and pretrained approximations. This approximator showcases the flexibility in approximating multihead attention modules.
